## Project: CLI Contact Book

You already know lists, dictionaries, and if statements — time to build stuff with it.

---

### What You're Building

A program that runs in the terminal and lets you manage contacts.

---

### Features

- Add a contact (name + phone number)
- View all contacts
- Search for a contact by name
- Quit the program

---

### The Menu (what the user sees)

~~~
Welcome to your Contact Book!
1. Add contact
2. View all contacts
3. Search by name
4. Quit
Enter your choice:
~~~


### Requirements

- Use a loop to keep the menu running until the user picks 4
